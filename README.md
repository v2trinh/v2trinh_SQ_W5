# Week 5 Side Quest — Maze with Animated Character and Coins

## Setup and Interaction Instructions

A single player maze game where the user must move the character and collect all three stars before they can exit at the end of the maze.

**Controls:** WASD to move.

Collect all 3 stars then reach the dark green exit tile to win.
